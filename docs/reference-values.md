# Wartości referencyjne współczynników biegu

Źródła: Novacheck 1998, Heiderscheit 2011, Souza 2016, Diaz 2019

## Kadencja [kroki/min]
- Biegacze rekreacyjni: 150–170
- Zaawansowani: 170–185
- Elita: 180–200
- Rekomendacja ogólna: celuj w 170–180 (zmniejsza obciążenie stawów)

## Czas kontaktu z podłożem (GCT) [ms]
- Wolny bieg (jogging): 250–350
- Bieg rekreacyjny: 200–280
- Szybki bieg: 160–220
- Elita: 150–200
- Im krótszy GCT → lepsza ekonomia biegu (do pewnej granicy)

## Czas lotu [ms]
- Typowy zakres: 80–150
- Brak fazy lotu = chód, nie bieg

## Duty factor
- Bieg: < 0.5 (jeśli > 0.5, to technicznie chód)
- Typowy bieg rekreacyjny: 0.35–0.45
- Sprint: 0.22–0.30

## Stride length [m]
- Zależy od wzrostu i prędkości
- Typowo: 1.0–1.5× wzrost biegacza przy biegu rekreacyjnym
- Overstriding: stopa ląduje >10 cm przed środkiem ciężkości

## Kąt kolana przy initial contact [°]
- Prawidłowy: 160–175° (lekko ugięte)
- Zbyt wyprostowane (>175°) = overstriding, ryzyko kontuzji
- Zbyt ugięte (<155°) = "siedzący" bieg, strata energii

## Pochylenie tułowia [°]
- Prawidłowe: 5–15° do przodu od pionu
- Zbyt pionowy (<5°) = brak wykorzystania grawitacji
- Zbyt pochylony (>20°) = nadmierne obciążenie dolnego odcinka kręgosłupa

## Vertical oscillation [cm]
- Dobry biegacz: 6–8 cm
- Biegacz rekreacyjny: 8–12 cm
- >12 cm = marnowanie energii na ruch pionowy

## Foot strike pattern
- Rearfoot (pięta): ~75% biegaczy rekreacyjnych
- Midfoot: ~20%
- Forefoot: ~5%
- Brak jednoznacznej rekomendacji "najlepszego" — ważniejsze jest overstriding

## Symetria L/P
- Różnica <5% = norma
- Różnica 5–10% = wymaga uwagi
- Różnica >10% = potencjalny problem (kontuzja, dysbalans)
